# Flight Alert Docs

This folder is for durable project guidance, not run logs or temporary evidence.

## Files

- `flightalert-performance-metrics.xlsx` - canonical performance workbook and optimization ledger.
- `code-organization.md` - package ownership and file-size guidance.
- `performance-debugging.md` - external probing/profiling guidance for optimization work.
- `snake-case-migration.md` - naming convention status and verification commands.

## Rules

- Keep detailed optimization workflow in `flightalert-performance-metrics.xlsx`.
- Keep generated screenshots, videos, traces, and frame outputs out of `docs/`.
- Update README/release notes when user-facing behavior changes.
